# ai-engineering-starter

Cloneable starter for AI-assisted engineering projects.

## Purpose

This repo provides a portable AI control structure that can be cloned as the starting point for a new project.

## Usage

1. Clone this repo into a new project repo
2. Remove the `meta/` folder
3. Fill in `docs/project/*`
4. Use `AGENTS.md` and `AGENT_RULES.md` as the active AI control layer

## Structure

- `AGENTS.md` — primary AI operating instructions
- `AGENT_RULES.md` — behavioral and anti-drift rules
- `docs/starter/` — portable control-system docs
- `docs/project/` — repo-specific docs to complete per project
- `docs/work-loops/` — ongoing execution records
- `meta/` — starter-design material; remove in cloned project repos
